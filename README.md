Учебный лендинг вымышленного аукциона.
Ссылка на репозиторий: https://github.com/polluxik/ono-tebe-nado-ad

## Языки и технологии

- **HTML**
- **CSS**
- Шрифт **Raleway** (локальные файлы в `fonts/`)

## Структура

- `index.html` — страница
- `styles/` — `global.css`, `style.css`
- `fonts/` — подключение шрифтов и файлы `.woff` / `.woff2`
- `images/` — картинки и иконки
